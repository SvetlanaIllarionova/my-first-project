
# Калькулятор оформителя банкетов

Профессиональный инструмент для расчёта стоимости оформления и проведения мероприятий.

## 🎉 Возможности

### Основные параметры
- Название мероприятия
- Дата проведения
- Количество гостей
- Площадь помещения

### Оформление & Декорации
- Расчёт стоимости арок
- Гирлянды и праздничное освещение
- Цветочные композиции
- Воздушные шары
- Прочие украшения

### Мебель & Оборудование
- Столы и стулья
- Звуковое и световое оборудование
- Сценическое оборудование
- Проектор и экраны

### Кейтеринг
- Расчёт стоимости питания на одного гостя
- Опция включения/исключения в общую смету

### Услуги персонала
- Дизайнер/оформитель (почасовая ставка)
- Монтажники (дневная ставка)
- Официанты и персонал обслуживания

### Дополнительные расходы
- Доставка материалов
- Страховка оборудования
- Непредвиденные расходы
- Автоматический расчёт наценки и прибыли

## 📊 Автоматический расчёт

Калькулятор автоматически вычисляет:
- Итоговую себестоимость мероприятия
- Сумму наценки и прибыли
- Финальную стоимость
- Стоимость на одного гостя

## 🚀 Как использовать

1. Откройте файл `banquet-calculator.html` в любом браузере
2. Заполните необходимые параметры мероприятия
3. Введите стоимость услуг и материалов
4. Калькулятор автоматически рассчитает итоговую смету
5. Используйте кнопку "Печать" для создания документа

## 💾 Функции

- **Реалтайм расчёты** - результаты обновляются при любом изменении
- **Гибкие параметры** - настройте любые значения под ваши услуги
- **Печать сметы** - создавайте профессиональные документы
- **Мобильная версия** - адаптивный дизайн для всех устройств
- **Форматирование валюты** - корректное отображение рублей
